# ExplApp Mirror Cast

نسخة HTML / PWA مخصصة للآيفون لإدارة إرسال الصور والفيديو والملفات والروابط للشاشات والرسيفرات.

## النسخة الحالية

- ملف واحد: `index.html`
- يعمل من Safari على iPhone
- يمكن إضافته إلى الشاشة الرئيسية
- اختيار عدة صور وفيديوهات وملفات دفعة واحدة
- قائمة تشغيل ومعاينة داخلية
- مشاركة الملف الحالي حسب ما يسمح به iOS
- صفحة أجهزة محفوظة وتجهيز لدعم Anyview Cast لاحقًا

## ملاحظات مهمة

- iPhone لا يدعم Miracast / Anyview Cast كمرآة شاشة كاملة من HTML.
- هذه النسخة تستهدف تجربة تشبه Tubio لكنها أبسط وأقوى في إدارة الملفات.
- دعم أعمق لـ Anyview Cast يحتاج نسخة Native أو تحليل بروتوكول الجهاز.
